## insertComments.vim


## installation
```vim
```

## Abstract
Insert comments into a javascript file.
Javascript file : '*.js' or '*.jsx'

## Command
Default command is <Ctr + i> (normal mode).

## Comments
Format is like this.
```
/**
  Author: ***
  CreatedAt: ***

  ClassName: ***
    Summary: ***
    Constructor: ***
        id: ***
        name: ***

  ClassName: ***
    Summary: ***
    Constructor: ***
        id: ***
        name: ***

*/
```
